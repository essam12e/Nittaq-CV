# نطاق | سي في | Nittaq | CV

نسخة موقع كاملة ومضبوطة بنفس المود الرقمي الفخم، مع دعم:
- رابط مباشر للصورة الشخصية
- رفع محلي للصورة الشخصية
- رابط مباشر للصور والفيديوهات والمواقع داخل الأعمال
- رفع محلي للصور والفيديوهات داخل الأعمال، مع دعم إضافة المواقع كرابط مباشر

## الملفات
- index.html
- styles.css
- app.js
- vercel.json
- README.md
- .gitignore

## المميزات
- عربي / إنجليزي
- صورة شخصية بمقاس عرض 1:1 وتظهر كاملة
- لوحة تحكم محلية
- مشاركة تنسخ رابط الصفحة الحالي
- جاهز للنشر على GitHub وVercel
- لا يحتاج Build tools أو مكتبات إضافية

## النشر على Vercel

- Build Command: فارغ
